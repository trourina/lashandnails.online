## ğŸ” SEO Configuration

### LocalBusiness Schema

Configured in `nuxt.config.ts` with:
- Business name and description
- Full address and geo coordinates
- Phone number
- Opening hours
- Price range

### Per-Page SEO

Each page has localized SEO meta tags:
- `title` - Page title
- `description` - Meta description
- `ogTitle` - Open Graph title
- `ogDescription` - Open Graph description

### Automatic Features

- Canonical URLs (auto-generated by @nuxtjs/seo)
- Sitemap.xml (auto-generated for all locales)
- robots.txt (auto-generated)
- hreflang tags (auto-generated for multilingual pages)
